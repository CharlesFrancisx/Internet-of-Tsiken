Internet of Tsiken

Monitor and protect your poultry with intelligent temperature control and real-time predator detection, all from a dedicated mobile application.

Download Learn More

Key Features

Intelligent tools for modern poultry farming.

Real-Time Monitoring

Track brooder temperature, humidity, and environmental conditions in real-time via ESP32 sensors.

Predator Alert System

Receive instant mobile notifications when predators are detected near the brooder via a YOLOv8n model on a Raspberry Pi.

Mobile Control Hub

Manage settings, view live data, and access historical analytics through a dedicated React Native mobile app.

System Requirements

What you'll need to run the application.

Hardware

  • ● Raspberry Pi 5 (or equivalent)
  • ● ESP32 Microcontroller
  • ● DHT22 Sensor (Temperature/Humidity)
  • ● 5V Relay Module & Heating Element
  • ● Pi Camera Module

Software

  • ● Android Device (OS 8.0 Oreo or newer)
  • ● Python 3.9+ on Raspberry Pi
  • ● PyTorch and OpenCV libraries
  • ● Arduino IDE for ESP32 programming
  • ● Stable Wi-Fi Connection

Academic Context & Team

Project Overview

Internet Of Tsiken is a capstone project demonstrating the integration of IoT hardware, computer vision, and mobile application development for agricultural automation. The system addresses real-world poultry farming challenges through intelligent brooder management and predator detection, evaluated against the ISO/IEC 25010 standard for software quality.

Technology Stack

  • •Frontend: React Native (Expo)
  • •Computer Vision: YOLOv8n
  • •Hardware: Raspberry Pi 5, ESP32

Evaluation Standards

  • •Framework: ISO/IEC 25010
  • •Focus: Functional Suitability, Usability
